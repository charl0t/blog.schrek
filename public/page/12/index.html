Hugo shortcode

2023-05-10 Web

Maj 15/05/2023: Ajout des icones personalisées des bandeaux. La syntaxe Markdown a des limites, Hugo utilise des « bouts de code » pour passer ces limites. L’inconvénient c’est de retenir les balises. L’avantage est d’intégrer des fonctions au sein même du contenu, sans toucher au template. Shortcode interne Vous trouverez pleins d’exemple sur le site officiel. https://gohugo.io/content-management/shortcodes/ shortcodes personnalisés Des icônes On va se servir des icônes du framework du site. https://getuikit.com/docs/icon Pour afficher une icône en HTML c est: <span uk-icon="icon: check"></span>
