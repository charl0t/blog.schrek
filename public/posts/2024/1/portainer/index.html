Portainer

C’est une application web qui permet de gérer docker via son navigateur.

Installation

Comme on a installé Docker précédemment, ca va aller très vite. Création d’un volume persistent Docker pour garder les données

$ sudo docker volume create portainer_data

Installation de Portainer

$ sudo docker run -d -p 9000:9000  --name portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer-ce:latest

Explications: run exécuter un contenaire -d En tache de fond -P port d’ecoute --restart=always Redémarre automatiquement le contenaire https://docs.docker.com/config/containers/start-containers-automatically/ -v Volume docker portainer/portainer-ce:latest Type de contenaire à exécuter

Plus de détailles ici https://docs.docker.com/engine/reference/commandline/run/

On liste les contenaires.

docker ps

Pour y acceder on utilise le port 9000, par exemple en local 127.0.0.1:9000
