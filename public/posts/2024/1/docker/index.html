    Docker avec un RaspberryPi 5

    2024-01-10 RaspberryPi
    Temps de lecture: 0 minutes.

    Cet article fait partie de la série: RaspberryPi


    Table des matières

    • Docker:
    • Installation:
    • Liens

    Docker:

    Docker est une plateforme permettant de lancer certaines applications dans des conteneurs. C’est différent des machines virtuels et ca utilise moins de ressource.

    Installation:

    On fait la mise a jour du système, cela permet de partir propre.

    $ sudo apt update
$ sudo apt upgrade

    Installation a l’aide du script officiel.

    $ curl -fsSL https://get.docker.com -o get-docker.sh
$ sudo sh get-docker.sh

    Apres quelques minutes, on regarde si ca tourne.

    $ sudo systemctl status docker

    On ajoute l’utilisateur pi a docker.

    $ sudo usermod -aG docker pi

    L’installation ne pose pas de problèmes particuliers.
