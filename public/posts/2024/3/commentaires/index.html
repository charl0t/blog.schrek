Le systeme de commentaires Cactus ne me satisfait plus, j’ai décidé de tester Disqus.

Installation

Il suffit de suivre le tutoriel sur le site d’Hugo

Rien de bien compliqué, c est un shortcode interne a Hugo.

Il faut deja creer un compte sur le site de Disqus.

On va récupérer le shortname dans la partie admin de Disqus

Dans notre fichier de configuration du site config.toml, on ajoute Disqus.

[services]
  [services.disqus]
    shortname = 'schrek-fr'

Pour l’intégration dans le site, j’ai choisi de placer le code dans le fichier /themes/schrek/layouts/_default/single.html

{{ template "_internal/disqus.html" . }}

En local ca ne marche pas, il faut envoyer les fichier sur un serveur.
